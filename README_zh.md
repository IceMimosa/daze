## 介绍

`Daze.js` 是一款高效、高扩展性、功能强大的 `Node.js` 服务端 Web 开发框架。

## 安装

```bash
$ npm install -g @dazejs/cli
```

## 快速开始

```bash
$ daze create example
$ cd example
$ npm start
```

访问 `http://localhost:8080` 进行预览
